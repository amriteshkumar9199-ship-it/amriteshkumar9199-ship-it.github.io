Amazon Associates Account बनाएं (Free)

affiliate-program.amazon.in पर जाएं और account बनाएं। यह बिल्कुल free है।

🔗 affiliate-program.amazon.in → Sign Up → Affiliate ID मिलेगी
2

Affiliate Links कैसे बनाएं (API के बिना)

किसी भी Amazon product की link लें और उसमें अपनी ID add करें:

amazon.in/dp/PRODUCT_ID?tag=AAPKI-ID

या Amazon Associate dashboard में "SiteStripe" tool से सीधे link copy करें। इस website के Admin Panel में URL डालें — tag automatic लगता है।

3

Amazon Product Advertising API — कब मिलेगी?

API तभी मिलती है जब आपके affiliate account में 3 qualified sales पूरी हों (180 दिन के अंदर)।

  • पहले manually products add करें (Admin Panel से)
  • 3 sales होने के बाद API ke liye apply करें
  • API मिलने के बाद products automatically fetch होंगे
📌 अभी करें: Manual products add करें → Traffic बढ़ाएं → Sales आएंगी → API मिलेगी
4

Traffic & Earning बढ़ाने के तरीके

  • Telegram Channel: हर deal Telegram पर share करें — सबसे fast growth

5

Earning कब और कैसे मिलती है?

  • Amazon हर महीने की 60 दिन बाद payment करता है
  • Minimum payout: ₹1,000 (Bank Transfer)
  • Payment सीधे आपके Bank Account में आती है
  • Admin Panel → Dashboard में daily earnings track करें
  • Amazon Associates dashboard से exact numbers copy करके Admin में enter करें
💡 Example: 100 clicks/day → ~5 purchases → avg ₹2000/purchase → ₹100 commission/day → ₹3,000/month से शुरुआत
6

Bank + PAN + Tax Details

  • Amazon Associates में Bank Account और PAN जरूरी है
  • TDS: अगर yearly income ₹30,000+ है तो TDS कटता है
  • Form 16A Amazon देता है — CA को दिखाएं
  • GST की जरूरत नहीं अगर आप individual हैं
